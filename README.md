# Pro3x Business Dashboard

Archived legacy project started in 2012, published solely as a code sample and as a basis for discussion.  
Originally developed as a simple POS application for issuing legally compliant fiscal invoices (per local regulatory requirements), it provided essential features for small business operations, including:

- product and item management  
- customer records  
- support for multiple cash registers  
- multi-location setup  
- multiple user accounts with access control  
- shift-based sales tracking per user  
- basic business reporting
